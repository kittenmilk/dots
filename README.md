# dotfiles

neko's i3-gaps setup

## details

wm: i3-gaps\
colors: catppuccin\
font: gohufont\
bar: polybar

## install

1. install paru
2. run `sh -c "$(curl -fsLS chezmoi.io/get)" -- init --apply kittenmilk/dots`
3. change your shell to zsh
4. run `discocss` to inject theme (you can start discord normally after you've injected it once)
